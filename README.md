# Awesome Native macOS Apps

More and more macOS apps are being shipped as web apps wrapped in webviews. They are convenient to build, but often come with heavy bundles, high memory usage, slower performance, and interfaces that do not quite feel at home on macOS.

This project exists for people who still care about native Mac software: apps that are fast, thoughtful, energy-efficient, and built with platform-first technologies such as Swift, Objective-C, AppKit, and SwiftUI.

The curated app library now lives on the website:

## [awesome-native-macos-apps.pages.dev](https://awesome-native-macos-apps.pages.dev)

Use it to discover native macOS apps worth installing, browse by category, and share the collection with other Mac users.

## About

Awesome Native macOS Apps is a curated library of apps that look, feel, and behave like they belong on the Mac. The goal is simple: make it easier to find great native alternatives before reaching for another heavy webview-based desktop app.

## Contributing

If you know a native macOS app that deserves a place in the collection, please open an issue or pull request.
